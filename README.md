# API Test Generator

This tool automatically creates test cases for any API. Give it an API description file, and it will generate ready-to-run tests.

## What It Does
- Reads API specifications (OpenAPI/Swagger files)
- Generates test cases for every endpoint
- Checks for errors and edge cases
